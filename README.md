# Статический сайт - портфолио

Репозиторий содержит отчеты и исходный код лабораторных работ по курсу программирования. Сайт с подробной документацией и разбором задач доступен по ссылке:

**[Сайт на github](https://kolyada-daria.github.io/portfolio/)**
**[Сайт на sourcecraft](https://portfolio-kdd.sourcecraft.site/portfolio-sc/)**

---

## Структура проекта


* `my-project/`
  * `docs/` — исходные файлы документации
    * `assets/` - ресурсы
    * `labs/` - md файлы с отчетами по лабораторным работам
    * `about.md` - об авторе
    * `index.md` - главная
  * `mkdocs.yml` — конфигурация сайта документации
  * `requirements.txt` - управление зависимостями


---

## Выбор темы: Material
Для оформления портфолио выбрана тема Material, так как она предлагает лучший набор инструментов для работы с техническим контентом.

* Поиск индексирует все лабораторные работы и позволяет найти нужный фрагмент кода или термин по всей базе отчетов сразу.

* Тема автоматически подсвечивает синтаксис кода, добавляет нумерацию строк. Это избавляет от ручного форматирования.

* Использование специальных раскрывающихся блоков (details, info, note) позволяет структурировать отчет.

* Навигация: справа всегда отображается динамическое оглавление страницы. Оно подсвечивает, в каком разделе отчета вы находитесь в данный момент, и позволяет быстро переключаться между задачами.

Адаптивность: Интерфейс автоматически подстраивается под экран (смартфон или ПК), сохраняя читаемость таблиц и схем без горизонтальной прокрутки.
