# Лабораторная работа №5: Регрессия с применением Scikit-Learn

### Цель работы
Разработка и комплексная оптимизация модели машинного обучения для решения задачи регрессии (предсказания стоимости недвижимости) с целью достижения максимальной точности прогноза и минимизации среднеквадратичной ошибки $RMSE$.

### Задание

* Заполнить пропуски в борде, дополнив кодом ячейки с соответствующим комментарием/заголовком.
* Выполнить самостоятельную работу, предложенную в конце борда.
* Исследовать параметры модели Random Forest и постараться получить ещё более низкое значение ошибки для этой модели.
* Исследовать (возможно с помощью LLM), каким образом обученную модель можно интегрировать с веб-сервисом, реализованным с помощью веб-фреймворка Flask / Django / FastAPI и описать тезисно пошаговый алгоритм, что необходимо для этого сделать.
