# Лабораторная работа №4: Классификация с применением Scikit-Learn

### Цель работы
Исследование и сравнительный анализ эффективности современных алгоритмов машинного обучения в задаче бинарной классификации. 
Освоение полного цикла разработки от предварительной обработки данных и борьбы с переобучением до оценки моделей с помощью метрики ROC-AUC.

### Задание

1. Заполнить пропуски в борде, дополнив кодом ячейки с соответствующим комментарием.
2. Исследовать другие модели для реализации классификации значений. По возможности построить более точную модель.
3. Исследуйте (опционально с помощью LLM) каким образом обученную модель можно интегрировать с веб-сервисом, реализованным с помощью веб-фреймворка Flask / Django / FastAPI и опишите тезисно пошаговый алгоритм, что необходимо для этого сделать.
